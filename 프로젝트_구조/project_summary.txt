# 상품 관리 앱

## 사용 기술
- SwiftUI
- IOS 26.2

## 기능
- 상품 목록 조회(다수) getProducts(name: "")  # name이 공백일 때는 데이터를 전부 불러옴
- 상품 상세 조회(단일) getProduct(id: String)
- 상품 추가 createProduct(name: String, description: String, price: Int)
- 상품 수정 updateProduct(id: String, name: String, description: String, price: Int)
- 상품 삭제 deleteProduct(id: String)
- 상품 이미지 조회 getProductImage(productId: String)
- 상품 이미지 업로드 uploadProductImage(productId: String, productImage: String)
- 상품 이미지 삭제 deleteProductImage(productId: String)

## 모델
Product
- id: String
- name: String - 1 ~ 100
- description: String - 1 ~ 500
- price: Int - 1 ~ 9999999

ProductImage(이미지는 상품 하나에 1개만 가능, 상품 이미지가 필수는 아님) 
- productId: String
- imageUrl: String(Base64)

사용할 아키텍처 
- MVVM
- Redux

사용할 라이브러리 
- ReSwift
- RxSwift
